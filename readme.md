# 🛒 E-Commerce Web Application

A full-stack e-commerce platform built using the MERN stack that enables users to browse products, manage their cart, place orders, and securely authenticate. The application also includes an admin dashboard for managing products and orders.

## 🚀 Features

### User Features

* User Registration & Login
* JWT-based Authentication
* Browse Products
* Search Products
* Add/Remove Items from Cart
* Place Orders
* View Order History
* Responsive Design

### Admin Features

* Add New Products
* Delete Products
* Manage Orders
* View Customer Purchases

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT (JSON Web Token)

### Other Tools

* Cloudinary (Image Storage)
* Razorpay / Stripe (Payment Gateway)
